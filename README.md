### Key Features <a name="key-features"></a>

- List all books
- List all music albums
- List all movies
- List of games
- List all genres (e.g 'Comedy', 'Thriller')
- List all labels (e.g. 'Gift', 'New')
- List all authors (e.g. 'Stephen King')
- List all sources (e.g. 'From a friend, 'Online shop')
- Add a book
- Add a music album
- Add a movie
- Add a game
